
# HOW IT WORKS

TEXT OPERATOR is an AI-powered knowledge instrument designed for reading, searching and composing large collections of documents.

Rather than replacing the author, it augments the research and writing process by making information searchable, interconnected and reusable.

---

# Overview

TEXT OPERATOR follows a simple workflow:

```
Import
    ↓
Index
    ↓
Search
    ↓
Explore
    ↓
Compose
    ↓
Export
```

Each step builds on the previous one.

---

# 1. Import

Documents are imported into the library.

Supported formats include:

- Markdown
- PDF
- EPUB
- TXT
- DOCX

The original files remain untouched.

---

# 2. Index

Every document is analyzed.

TEXT OPERATOR extracts:

- title
- author
- metadata
- chapters
- paragraphs
- keywords
- embeddings

This creates a semantic index of the library.

---

# 3. Search

Instead of searching words, users search ideas.

Examples:

> books about systems thinking

> all references to attachment theory

> compare Jung and Bateson

Results are ranked by semantic similarity.

---

# 4. Explore

Every result becomes an entry point.

Users can:

- open the source
- inspect context
- discover related passages
- follow semantic links
- navigate through the knowledge graph

Reading becomes exploration.

---

# 5. Compose

Selected passages can be collected inside a workspace.

Users build:

- notes
- outlines
- articles
- essays
- books

The workspace always keeps references to the original sources.

---

# 6. Export

Projects can be exported as:

- Markdown
- PDF
- DOCX

The original bibliography is preserved.

---

# Local AI

TEXT OPERATOR is designed to work with local language models whenever possible.

Personal libraries remain under the user's control.

---

# Design Principles

- Human-first
- Meaning-first
- Local-first
- Search before generation
- References over hallucinations
- Beautiful interfaces
- Deep work

---

# Product Philosophy

TEXT OPERATOR is not a chatbot.

It is a knowledge instrument.

Its purpose is to help people understand more, connect ideas and write better.
